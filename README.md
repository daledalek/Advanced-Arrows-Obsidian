# Advanced-Arrows
plugin to replace text with unicode arrows

automatically enabled text replacements:
.do → ↓
.up → ↑
.lr → ↔
.ud → ↕
.nw → ↖
.ne → ↗
.se → ↘
.sw → ↙
.rol → ⇄
.lor → ⇆
.uod → ⇅
.zz → ↯
.acw → ↺
.cw → ↻

optional text replacements (enabling compatibility with Smart Typography plugin):
-> → → 
<- → ←
